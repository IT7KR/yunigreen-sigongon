## 1. 문서 개요

### 목적

본 문서는 유니그린 SaaS 플랫폼(시공ON) 개발 과정에서 진행된 총 5회의 미팅(사전 미팅 3회, 킥오프 미팅 2회)에서 확정된 모든 결정사항을 **도메인별로 분류하여 단일 레퍼런스**로 제공합니다. 개발팀이 구현 시 참조해야 할 확정 사항, 미확정 사항, 그리고 결정 변경 이력을 한눈에 파악할 수 있도록 합니다.

### 활용 방법

- **개발 착수 시**: 해당 도메인의 확정 사항 테이블을 기준으로 구현
- **기능 스펙 문서(11_FEATURE_SPEC.md)와 불일치 발견 시**: 본 문서의 결정사항이 최종 기준 (미팅 확정 > 스펙 문서)
- **미확정 사항 확인**: 섹션 6의 교차 참조표에서 현재 상태 확인

### 관련 문서

| 문서 | 설명 |
|------|------|
| `05_DOCUMENT_WORKFLOW.md` | 프로젝트 생애주기별 문서 워크플로우 |
| `06_PENDING_REVIEW_ITEMS.md` | 미확정/검토 필요 항목 상세 |
| `11_FEATURE_SPEC.md` | 기능 명세서 (기능 ID 정의) |

## 8. 업데이트 규칙

1. **미팅 직후**: 회의록에서 결정사항 추출
2. **본 문서 반영**: 해당 도메인 테이블에 결정사항 추가/수정
