> 2025년 1월 17일 메일 발송

## 개요

프로젝트 개발 진행을 위해 클라이언트에게 요청한 준비사항 목록입니다.

---

## 체크리스트

- [ ] OpenAI API Key
- [ ] Gemini API Key
- [ ] 서비스명 (정식/영문/슬로건)
- [ ] 공식 로고 (PNG / SVG는 권장)
- [ ] 파비콘 (원본 이미지)

- [ ] 온라인 미팅 일정 회신

---

## 상세 요청 내역

### 1. OpenAI API Key

**용도**
- 단가표 및 시공기준 문서에서 관련 정보를 검색하는 RAG 시스템에 사용
- 텍스트 의미 기반 검색을 통해 정확한 자재/공법 정보 검색

**준비 방법**
1. OpenAI 플랫폼 접속 (platform.openai.com)
2. 회원가입 또는 로그인
3. 좌측 메뉴에서 "API Keys" 클릭
4. "Create new secret key" 클릭
5. 키 이름 입력 (예: yunigreen-production)
6. 생성된 키 복사 (sk-xxxxx 형태)

**주의사항**
- 키는 생성 시 한 번만 표시되므로 반드시 복사 필요
- 결제 수단 등록 필요 (선불 크레딧 충전 방식, $20 정도면 개발 단계 충분)

---

### 2. Gemini API Key

**용도**
- 현장에서 촬영한 사진을 AI가 분석하여 누수 원인과 위치 판단
- 분석 결과를 바탕으로 적합한 자재와 시공 방법 추천

**준비 방법**
1. Google AI Studio 접속 (aistudio.google.com)
2. Google 계정으로 로그인
3. 좌측 메뉴에서 "Get API Key" 클릭
4. "Create API key" 클릭
5. 프로젝트 선택 또는 새 프로젝트 생성
6. 생성된 키 복사 (AIzaSy... 형태)

**예상 비용**
- 무료 사용량(Free Tier)으로 개발/테스트 진행 가능
- 운영 단계에서는 사용량에 따라 유료 전환 필요 가능

**참고사항**
- 프로덕션 운영 시 요청 제한 해제 등을 위해 유료 플랜 사용 권장

---

### 3. 서비스명 최종 확정

**용도**
- 웹사이트 타이틀, 로그인 화면, 견적서/계약서 PDF 등에 표시

**확정 필요 항목**
| 항목 | 예시 | 입력값 |
|-----|-----|-------|
| 정식 서비스명 | 유니그린 | |
| 영문 서비스명 | Yunigreen | |
| 슬로건/설명 | AI 누수진단 시스템 | |

---

### 4. 서비스 공식 로고

**용도**
- 웹/모바일 UI 상단, 로그인 화면 등 서비스명이 표기되어야 할 부분에 사용

**필요 파일**
| 형식 | 규격 | 필수 여부 |
|-----|-----|:--------:|
| PNG (투명 배경) | 최소 512x512px | 필수 |
| PNG (흰색 버전) | 다크모드용 | 권장 |
| SVG | 벡터 | 권장 |

**참고사항**
- 가로형 로고 권장
- 투명 배경 PNG 필수

---

### 5. 파비콘

**용도**
- 브라우저 탭과 북마크에 표시되는 작은 아이콘

**필요 파일**
- 원본 이미지 (512x512px 이상 정사각형)

**참고사항**
- 로고의 심볼(아이콘) 부분만 사용 권장
- 텍스트는 작은 크기에서 잘 보이지 않음
- 원본 이미지만 제공 시 파비콘 파일 자동 생성 가능

---

### 6. 앱 서비스명 (현장용)

**용도**
- 모바일에서 "홈 화면에 추가" 시 표시되는 이름

**확정 필요 항목**
| 항목 | 예시 | 입력값 |
|-----|-----|-------|
| 현장앱 앱명칭 | 유니그린 현장 | |

**참고사항**
- 너무 길면 말줄임표(...)로 잘림

---

### 7. 앱 아이콘

**용도**
- 모바일 홈 화면 아이콘, 앱 전환 화면에 표시

**필요 파일**
| 크기 | 용도 | 필수 여부 |
|-----|-----|:--------:|
| 512 x 512px | 스플래시, 고해상도 | 필수 |
| 192 x 192px | 홈 화면 아이콘 | 필수 |

**참고사항**
- PNG 형식, 정사각형
- 로고의 심볼만 사용 권장 (텍스트 제외)
- 배경색은 브랜드 컬러 사용 권장
- 모서리는 정사각형 (OS에서 자동으로 둥글게 처리)
- 가장자리에 10% 정도 여백 권장

---

### 8. 브랜드 컬러 확정

**용도**
- 전체 UI 테마(버튼/링크/강조색) 및 기타 디자인에 적용

**현재 설정값**
- 메인 컬러: #14b8a6 (Teal)

**확정 필요 항목**
| 항목 | 값 |
|-----|---|
| 메인 컬러 (Primary) | #______ |
| 보조 컬러 (Secondary) | #______ (선택) |

**참고사항**
- 로고에 사용된 색상과 통일 권장
- HEX 코드로 전달 (예: #14b8a6)
- 색상 견본 이미지 또는 레퍼런스 웹사이트도 가능
