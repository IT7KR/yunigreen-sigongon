[확정/합의 사항]

1) 원가계산서 기능 추가
- 견적서 생성 시 원가계산서 탭을 별도로 제공해야 함
- 원가계산서에 포함되는 주요 항목:
  - 직접 노무비, 재료비, 기계경비 (집계표에서 연동)
  - 산재보험료, 고용보험료, 건강보험료 (노무비 기준 요율 적용)
  - 안전관리비, 이윤, 일반관리비, 기타경비
- 요율 편집 가능 항목: 이윤, 일반관리비, 기타경비 → 수기 수정 지원
- 요율 고정 항목: 산재·고용보험 등 법정 요율 (연 2회 상반기/하반기 갱신 대상)
- 기존 보험 요율 관리 방식과 동일하게 요율 수정 기능 적용 예정
- 건강보험료는 공사 기간 31일 이상일 때만 적용

원가계산서 흐름 (엑셀 기준):
  단가대비표 → 일이대가 → 내역서(재료비/노무비/경비) → 집계표 → 원가계산서
- 내역서 합계(재료비·노무비·경비)가 집계표로 합산되고,
  원가계산서에서 해당 값에 요율을 적용해 최종 단가 산출

2) 원가계산서 다운로드 방식
- 웹에서는 견적서 내 원가계산서 탭으로 표시
- 다운로드 시 견적서와 원가계산서를 하나로 합쳐서 받을 수 있어야 함 (기본 동작)
- 개별 다운로드 옵션도 고려

3) 원가계산서 공개 여부 제어
- 관공서(공공) 발주: 원가계산서 무조건 포함 (법규 의무)
- 민간 소액 발주: 원가계산서를 숨기고 발행 가능 → 사용자가 임의로 선택하는 방식 채택
- 금액 기준 자동 제어가 아닌, 사용자 직접 토글/선택 방식으로 구현

4) 견적서 내 노무비 항목 처리
- 방수공·도장공·특별인부 등 직종 분류는 견적서 작성 목적의 표준 양식으로,
  실제 근로자 등록 구분과는 무관
- 적산 정보에 없는 공종(미등재 단가)은 현장 소장이 직접 금액 입력
- 견적서 작성 시 항목 추가/삭제/수정 가능하도록 구현

5) 자재 발주 — AI 연계 로드맵
- 이번 1차 범위: 수기 입력 기반 자재 발주 기능 유지
- AI 추천 → 자재 발주 자동 연동: 2차 고도화 과제로 이관
  - AI 추천 공법/자재 결과에서 "발주하기" 버튼으로 발주 탭 이동
  - 확인/승인 워크플로우 포함한 세부 구현은 고도화 시 논의
- AI 추천 자재 리스트 자동 등록까지는 1차에서 가능한 범위로 검토

6) AI 진단 사진 매수 최종 확정

- 운영 중 10장 미만으로 조정 가능하며, 추후 변경도 어렵지 않음
