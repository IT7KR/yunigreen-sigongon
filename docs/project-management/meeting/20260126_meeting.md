# 미팅 회의록 (2026-01-26)

- 목적: 스토리보드 초안 공유, 회원가입/결제/권한 구조 구체화, 적산정보 관리 논의

  - IT7: 이중호 대표, 최대민 개발자
  - 유니그린개발: 이시대 대표

## 확정/합의 사항

### 1) 결제(PG) 확정
- 나이스페이먼트: 유니그린개발 측 사용 중이 아닌 것 확인(나이스페이먼트 프로덕션 오류 빈번)
- 토스페이먼츠로 진행 확정

### 2) 회원가입 중복 제한
- 동일 휴대전화번호로 중복가입 제한
- 동일 사업자등록번호로 중복가입 제한
- 중복 시 안내 메시지 표시("이미 가입된 번호/사업자번호입니다")

### 3) 비밀번호 재설정
- 휴대전화 기반 인증번호(OTP) 방식으로 확정
- 흐름: 아이디 입력 → 등록된 휴대전화로 인증번호 발송 → 인증 완료 후 새 비밀번호 설정

### 4) 적산정보 관리
- PDF 업로드 방식으로 적산 데이터 관리
- 버전 관리 지원(업로드 이력 보존)
- 업로드 권한: 최고관리자(유니그린개발)만 가능
- PDF 파싱 처리시간: 수십 초~수 분 예상(비동기 처리)

### 5) 계정/권한 구조 확정
- **최고관리자(유니그린개발)**: 전체 시스템 관리 전용, 고객사 데이터 전체 조회 가능
- **고객사 대표**: 1개 고객사당 대표 1명(회원가입 시 등록)
- **고객사 실무자**: 선택 사항, 대표가 이메일 초대로 추가
- **현장소장**: 복수 가능, 대표가 이메일 초대로 추가
- 실무 계정(현장소장)과 관리 계정(대표/실무자) 분리 운영

### 6) 고객사 관리 화면
- 고객사 목록에 요금제 금액 표시 컬럼 추가
- 고객사 상세 화면에 구독 상태, 결제 이력, 커스텀 무료기간 설정 기능 포함
- 최고관리자 전용 화면

### 7) 고객사 실무자 정보
- 회원가입 시 실무자 정보 입력은 필수가 아님
- 마이페이지에서 추후 입력 가능
- 실무자 추가는 고객사 대표가 초대 방식으로 진행

### 8) 사용자 초대 방식
- 고객사 대표가 이메일 초대로 실무자/현장소장 추가
- 직접 가입(회원가입 페이지)으로는 실무자/현장소장 등록 불가
- 초대 흐름: 대표가 이메일 입력 → 초대 링크 발송 → 수신자가 링크 클릭 후 정보 입력/계정 생성

### 9) 일용직 근로자 가입
- 노무정보 등록 시 자동 회원가입 처리(현장소장 또는 대표가 등록)
- 근로자는 자동 생성된 계정으로 로그인 후 필수 서류 등록
  - 기초 산업안전보건교육 이수증
  - 신분증
  - 계좌정보 등
- 서류 미등록 시 근로 투입 제한

### 10) 개인정보 동의
- 일용직 근로자의 개인정보를 시스템에 등록·관리하기 위한 동의 절차 필요
- 구체적 동의 방식(서면/전자/앱 내)은 유니그린개발 측 추가 검토 후 제안 예정
- 법적 요건(개인정보보호법) 준수 필수

### 11) 데이터 격리
- 고객사 간 근로자 이력 완전 격리
- A 고객사에서 등록한 근로자 정보를 B 고객사에서 조회 불가
- 최고관리자(유니그린개발)만 전체 근로자 데이터 조회 가능

### 12) 프로젝트 분류
- 건설업 신고 분류 기준에 따른 카테고리 선택 기능 추가
- 프로젝트 생성 시 해당 분류 선택 필수

### 13) 작업일지 날씨
- 기상청 또는 케이웨더 API 연동 검토
- 대안: 수기 입력(사용자가 직접 날씨 선택)
- 최종 결정: 유니그린개발 측 추가 검토 후 확정 예정
