### 주요 논의 사항

#### 1. IT7 회사 소개

- 2011년 7월 창업, 15년차 개발사
- 170여개 프로젝트 수행 (공개 130여개, 비공개 포함)
- 앱/웹 플랫폼 개발 및 유지보수 전문
- 유사 프로젝트: 건설 현장 하자 점검 시스템, 머신러닝 기반 이미지 인식

#### 2. AI 기술 적용 방안

**AI 사진 분석**

- 현장 사진 촬영 → AI 분석 → 공종/시공법 추천
- 목표 정확도: 85% 이상
- 85% 정확도 기반 초안 생성 후, 잔여 15%는 현장 실무자 수기 보정 전제
- ChatGPT, Gemini API 모두 활용 가능
- Gemini 3 모델 출시로 비용 절감 가능

**RAG 시스템**

- 정답집(단가표, 시공기준) 데이터베이스 구축
- AI가 참조하여 답변 품질 향상
- 사용할수록 학습되어 정확도 개선

**기존 유사 프로젝트 사례**

- 문 인식 프로젝트: 2,200장 학습 데이터로 구현
- 광고 문구 생성 서비스
- 작가 글쓰기 보조 도구
- 나라장터 제안서 자동화: 나라장터 공공 API로 공고문 스크래핑 → HWP→PDF 변환 → GPT 기반 제안서 자동작성 (IT7 레퍼런스 사례, 본 프로젝트 범위 외)

#### 3. 견적서 자동화

**흐름**

```
현장 사진 촬영 → AI 기술진단서 → 견적서 자동 생성 → 시방서 연동 → 계약서
```

**적산정보(한국물가정보) 연동**

- PDF 형태로 제공되는 단가표
- 연 2회 업데이트 (단가만 변경, 공종은 유지)
- 카테고리 기반 검색으로 파싱 가능
- 엑셀 양식에 맞춰 자동 변환
- AI가 공종/시공법 초안을 제시하고, 수량은 담당자가 현장에서 보정
- 단가 계산은 적산정보 기준 데이터(엑셀/DB)로 처리 (AI 계산 미사용)

**음성 입력 지원**

- 현장에서 음성으로 공종 입력
- 음성 → 텍스트 → 항목 자동 선택
- 모바일 환경에서 편의성 향상

#### 4. 전자계약/서명

**일용직 근로계약**

- 일용직 근로자와 당일 계약/지급 프로세스
- 사인 이미지 기반 자체 전자서명 구현 가능
- 법적 효력은 계약 유형/운영 방식에 따라 별도 법무 검토 필요
- 공증 수준의 검증이 필요한 경우 외부 전자서명 서비스 연동 검토

**근로신고 양식**

- 콘텍스 등 연동은 제한적 (더존 등 과점 계약)
- 자체 양식 생성 및 다운로드로 대응

#### 5. 프로젝트 관리 기능

**공사별 프로젝트 관리**

- 새 프로젝트 생성 → 착공 → 계약 → 노임 관리
- 일용직 인력 등록 및 계약서 관리
- 프로젝트별 서류 다운로드 (견적서, 시방서, 계약서)

**서류 자동화**

- 견적서 → 시방서 → 계약서 흐름
- 공종 결정 시 관련 서류 자동 생성
- 한글파일 템플릿 기반 (틀은 고정, 내용만 변경)
- 공종 결정 → 표준 공법별 시방서 자동 로딩
- 수기 수정 가능한 에디터 제공 (로딩된 내용 편집 후 저장)

#### 6. 사진 관리

**모바일 실시간 촬영**

- 웹앱에서 카메라 직접 열어 촬영 → 즉시 서버 저장 (갤러리→업로드 2단계 불필요)
- 촬영 완료 시 별도 확인 없이 자동 업로드

**전/중/후 사진 분류**

- 촬영 시간 메타데이터(EXIF) 활용
- 시간 순서대로 자동 정렬
- 카카오톡 전송 시 EXIF 메타데이터 손실 가능 → 앱 내 직접 촬영 권장 (앱 촬영 시 EXIF 보존 확인)

**사진첩 자동 생성**

- 선택한 사진들 자동 콜라주
- 3열, 4열 등 레이아웃 선택 가능
- PDF 다운로드
- 사진 데이터 최소 3년 보관 요구

**중복 사진 제거**

- 유사도 기반 비교
- 동일 부위 판단 시 안내
- 단, 저장 공간은 크게 문제되지 않음 (이미지 압축)

#### 7. 안전관리 사진 인식

**안전관리비 증빙**

- 모자/안전화 착용 여부 AI 인식 가능
- 불가한 사진 업로드 시 안내 기능
- 안전화 상태(신품 여부)는 판단 어려움

#### 8. 개발 일정 및 배포 계획

**개발 기간**

- 목표: 150일 이내
- 배포 희망: 2026년 6-7월 (여름 성수기)

**사업 모델**

- SaaS 구독형 서비스
- 유니그린 자체 사용 후 피드백 반영
- 타 방수업체(1인 사업자 포함) 배포
- 사용 요금: 월 구독료 (1-3단계 플랜)

**운영 방식**

- 호스팅 포함
- 위클리 리포트 제공 (목요일)
- 개발 서버 공개로 진행 상황 확인
- 스토리지: 클라우드(S3 사용량 과금) 우선 검토, 필요 시 온프레미스 대안 검토

### 주요 논의 사항

#### 1. AI 샘플 시연 및 프롬프트 설명

- ChatGPT 4.1 mini 버전으로 누수소견서 샘플 작성
- 프롬프트 구성: 페르소나 설정, 공사개요, 현장평가 등 항목 목록화
- 예상 투입 자재별 소요량 추정 기능 포함
- 정확도 우려: 할루시네이션(허위 정보) 가능성 있음

#### 2. AI 정확도 및 학습 데이터 논의

- 학습 데이터 필요: 문 인식 프로젝트는 2,200장 사용
- 건물 유형(옥상, 외벽, 지하, 바닥, 천장)별 데이터 필요
- **현재는 참고 자료 수준으로 활용, 전문가가 수정 후 고객 제공**
- 고객이 직접 사진 올려서 소견서 자동 생성 기능도 가능
- 역방향 활용: 엔드 고객(건물주)이 누수 사진을 직접 촬영·전송 시, AI 소견서가 함께 생성되어 시공사와의 커뮤니케이션 효율화
- 데이터 축적 후 재학습으로 정확도 개선 가능 (비용 발생)

#### 3. Gemini vs ChatGPT 비교 결과

- Gemini: 이미지 판독 정확도 더 높음 (비주얼 분석 최고)
- ChatGPT: 텍스트 응답이 더 상세하지만 비용 높음
- Gemini 2.0 테스트 대비 Gemini 3 계열에서 정확도 향상 예상
- **결론: 이미지 판독은 Gemini 3 계열(개발 시점 최신 안정 버전) 우선 사용**

#### 4. 시스템 구성 명확화

```
사진 촬영 → [AI] 사진 판독/공법 제안 → [RAG] 적산정보 검색 → [ERP] 견적서 생성
```

- **AI 기능**: 사진 판독 + 공법 제안
- **RAG 기능**: 적산정보 PDF에서 자재/단가 검색 (AI)
- **ERP 기능**: 견적서 생성 (데이터베이스 기반, AI 아님)
- CSV 형태로 데이터 추출 → 엑셀/견적서에 삽입 가능
- AI/RAG 추출 결과는 실무자 검토·수정 후 최종 견적 확정
- 프롬프트 관리: 관리자 화면에서 AI 프롬프트 변경 가능 (운영 중 조정 가능)

#### 5. 적산정보 PDF 파싱 결과

- 파싱 성공: 구조화 잘 되어있음 (기계가 이해하기 좋은 구조)
- 380페이지 2개 파일 처리 완료
- 텍스트 기반 PDF가 이미지 기반보다 정확도 높음
- 키워드 검색 → RAG 적용 시 자연어 질의 가능

#### 6. 소견서 출력 형식

- 기존: 글만 출력
- 요청: 사진 + 기술 설명이 함께 PDF로 출력
- **구현 가능**: 사진과 해당 부위 설명을 함께 출력
- 요구 형식: 사진 영역 표시 + 옆에 기술 설명 병렬 배치
- 내용 간소화 필요: 현재 프롬프트 결과가 장황 → 필요 항목 규격을 유니그린 측에서 제공

#### 7. 전자계약 상세 논의

**기본 방향**

- 자체 전자계약(서명/도장 이미지, 타임스탬프 기록) 우선 적용

**공사도급계약**

- 전자서명법 확인 후 요건 충족 여부 판단
- 필요 시 외부 API(모두싸인 등) 연동 검토

**근로계약(일용직)**

- 자체 개발 가능, 외부 서비스 불필요
- 사인/도장 이미지 업로드로 구현
- 계약 시간 (년/월/일/시/분/초) 기록으로 법적 효력

#### 8. 외부 연동 제한 사항

**전문건설공제조합**

- API 미제공
- 스크래핑은 정보보안 위배 위험
- 보증서는 PDF 업로드 + 수기 입력으로 대응
- 보증서 원본: 보안 정책상 PDF 다운로드 불가 → 출력 후 스캔 필요
- 수기 입력이 더 효율적: 공사건당 보증서 1건이므로 직접 입력 권장
- 증빙용 PDF 별도 업로드 가능 (대조 목적)
- **프로젝트 내 '하자 관리' 탭 신설** 합의: 날짜, 공사명, 보증기간 등 관리

**기타**

- 홈택스 연동: 일반 사업자에게 API 미제공

**외부 연동 원칙**

- 공공데이터포털 공식 Open API만 사용 (스크래핑은 정보보안 위배)
- 나라장터: 공공데이터포털 제공 (사용 가능)
- 전문건설공제조합: 미제공 (연동 불가)

#### 9. 앱 개발 전략

**기능 범위**

- 현장 사진 촬영, 소견서 확인 등 간략한 기능
- 복잡한 기능은 PC 웹에서 처리
- 간편 견적서 작성: 현장에서 수기 입력 → 전자계약 직행 (개인사업자 민간 현장용)

**결제 전략**

- 앱 내 결제 시 수수료 30-40% (구글/애플)
- **웹에서 결제, 앱은 기능만 제공**

**플랫폼 전략**

- **안드로이드 우선** (현장 인력 대부분 삼성폰 사용)
