1. 현장별 일용신고명세서
( ex. ㅇㅇ중학교 방수공사 일용신고명세서 )
- 각 현장의 공사기간에 사용된 일용직 근로자들의 명단이 필요합니다.
- 공사 기간이 두개의 월에 걸쳐서 진행되는경우 N월, M월에 해당하는 현장별 일용신고명세서가 필요합니다.

예를들어 ㅇㅇ중학교 공사기간이 1/20~2/14 인 경우
        - 'ㅇㅇ중학교 방수공사 현장별 일용신고명세서(1월)'
        - 'ㅇㅇ중학교 방수공사 현장별 일용신고명세서(2월)'

이런식으로 2개의 월에 해당하는 현장별 일용신고명세서가 나와야합니다.
 엑셀파일로 다운받을 수 있어야합니다.



2. '현장별 일용신고 명세서'를 토대로 한 '근로복지공단 신고 양식' 자동채우기
- 이전에 보내드렸던 '근로복지공단 신고 양식'에 데이터를 '현장별 일용신고명세서'를 토대로 자동채우기 되어 엑셀로 다운받고싶습니다.
- 미팅에서 언급된 바와 같이 주민등록번호는 수기로 입력가능하게 나와도 될 거 같습니다.




3. 월별 일용노무비 지급명세서(월별 통합본)
( ex. 유니그린개발 2026년 1월 일용근로자 사용 명단)
- 각 월 전체 일수 동안 우리 회사에서 일용근로자를 얼마나 사용했는지 파악하기 위한 자료입니다. 해당자료를 통해 사업장 세무 신고를 진행합니다.
- 한 월에 현장이 여러개가 있을 수 있고, 한 사람을 두개의 현장에서 사용할 수 있기 때문에 통합본이 필요합니다.

예를들어 근로자 '홍길동'을 2026년 1월에 11~15일 동안 ㅇㅇ중학교에 투입했고
16~20일 동안 ㅇㅇ고등학교에 투입했다면, 현장별 일용신고명세서에는 각각의 현장에 기입되겠지만/ 1월 일용신고명세서 통합본에는 11일~20일 동안 일용근로를 했다고 합쳐서 보였으면 합니다.
