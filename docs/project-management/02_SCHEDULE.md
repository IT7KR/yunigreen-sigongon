### Phase 3: 개발 (3/3 ~ 5/15)

**목표**: 기능 개발 완료

**병렬 진행 전략**
- 3/3 ~ 3/23: 관리자 기능 + 공통 인프라 선개발
- 3/24 ~ 5/15: 사용자 화면 개발 (디자인 확정 후)

#### 3/3 ~ 3/23 (관리자 기능 우선)

| 항목 | 설명 |
|-----|------|
| 공통 인프라 | 인증, DB, API 기본 구조 |
| 관리자 - 프로젝트 관리 | CRUD, 목록/상세 |
| 관리자 - 견적서 관리 | 작성, 수정, PDF 생성 |
| 관리자 - 단가표 관리 | 업로드, 검색 |
| 관리자 - 사용자 관리 | 계정 관리 |

#### 3/24 ~ 5/15 (사용자 화면 개발)

| 항목 | 설명 |
|-----|------|
| 현장앱 - 홈 | 대시보드, 오늘의 일정 |
| 현장앱 - 프로젝트 | 목록, 상세, 현장 등록 |
| 현장앱 - AI 진단 | 사진 촬영, AI 분석 |
| 현장앱 - 현장 방문 | 방문 기록 작성 |
| 현장앱 - 견적 확인 | 견적서 조회/공유 |

**마일스톤**
- [ ] 관리자 기능 개발 완료 (3/23)
- [ ] 현장앱 개발 완료 (5/15)
- [ ] 통합 테스트 완료 (5/15)
