# 유니그린 SaaS - UX 라이팅 가이드

> 토스 UX 라이팅 원칙을 기반으로 한 유니그린 라이팅 가이드

## 핵심 가치 (Core Values)

| 가치 | 설명 | 예시 |
|-----|------|-----|
| **명확한 (Clear)** | 한 번에 이해할 수 있게 | ❌ 인증 실패 → ✅ 비밀번호가 틀렸어요 |
| **간결한 (Concise)** | 꼭 필요한 내용만 | ❌ 현재 진행 중입니다 → ✅ 진행 중 |
| **친근한 (Casual)** | 딱딱하지 않게 | ❌ 완료되었습니다 → ✅ 완료했어요 |
| **존중하는 (Respect)** | 강요하지 않게 | ❌ 필수로 입력하세요 → ✅ 입력해 주세요 |
| **공감하는 (Emotional)** | 상황에 맞는 감정으로 | ❌ 처리 완료 → ✅ 견적서를 보냈어요 |

## 라이팅 원칙

### 1. 해요체 사용

모든 문구는 해요체로 써요.

| Don't | Do |
|-------|-----|
| 저장되었습니다 | 저장했어요 |
| 확인하시겠습니까? | 확인할까요? |
| 입력해 주십시오 | 입력해 주세요 |

### 2. 능동적으로 쓰기

수동태보다 능동태를 써요.

| Don't | Do |
|-------|-----|
| 사진이 업로드됐어요 | 사진을 올렸어요 |
| 견적서가 생성됐어요 | 견적서를 만들었어요 |
| 프로젝트가 등록됐어요 | 프로젝트를 등록했어요 |

### 3. 긍정적으로 쓰기

부정형보다 긍정형을 써요.

| Don't | Do |
|-------|-----|
| 로그인할 수 없어요 | 비밀번호를 확인해 주세요 |
| 사진이 없어요 | 사진을 올려주세요 |
| 권한이 없어요 | 관리자에게 권한을 요청해 주세요 |

### 4. 캐주얼한 경어

과도한 경어는 쓰지 않아요.

| Don't | Do |
|-------|-----|
| 확인하시겠어요? | 확인할까요? |
| 이용하시겠어요? | 이용할까요? |
| 저장하시겠습니까? | 저장할까요? |
| OO님께 | OO님에게 |

### 5. 잡초 제거 (Weed Cutting)

의미 없는 단어는 빼요.

| Don't | Do |
|-------|-----|
| 현재 진행 중이에요 | 진행 중이에요 |
| 앞으로 예정이에요 | 예정이에요 |
| 혹시 문제가 있나요? | 문제가 있나요? |

### 6. 한자어 풀어쓰기

명사 나열보다 동사로 풀어 써요.

| Don't | Do |
|-------|-----|
| 진단 요청 완료 | 진단을 요청했어요 |
| 견적서 발행 불가 | 견적서를 발행할 수 없어요 |
| 인증 오류 발생 | 인증에 실패했어요 |

## 상황별 메시지 가이드

### 성공 메시지

```
✅ 저장했어요
✅ 사진을 올렸어요
✅ 견적서를 발행했어요
✅ 진단을 요청했어요
✅ 프로젝트를 만들었어요
```

### 에러 메시지

에러 메시지는 **원인 + 해결 방법**을 함께 알려줘요.

```
❌ 인증 실패
✅ 비밀번호가 틀렸어요. 다시 입력해 주세요.

❌ 파일 업로드 실패
✅ 사진 용량이 너무 커요. 10MB 이하로 올려주세요.

❌ 권한 없음
✅ 이 기능은 관리자만 쓸 수 있어요.

❌ 서버 오류
✅ 잠시 문제가 생겼어요. 다시 시도해 주세요.
```

### 확인 메시지 (CTA)

사용자에게 다음 행동을 명확히 안내해요.

```
✅ 견적서를 발행할까요? 발행하면 수정할 수 없어요.
✅ 프로젝트를 삭제할까요? 삭제하면 되돌릴 수 없어요.
✅ 진단을 요청할까요? 30초 정도 걸려요.
```

### 로딩/진행 상태

```
✅ AI가 사진을 분석하고 있어요...
✅ 견적서를 만들고 있어요...
✅ 저장하는 중...
```

### 빈 상태 (Empty State)

```
✅ 아직 프로젝트가 없어요. 새 프로젝트를 만들어보세요.
✅ 올린 사진이 없어요. 현장 사진을 올려주세요.
✅ 진단 결과가 없어요. AI 진단을 요청해 보세요.
```

## 에러 코드별 메시지

| 코드 | 메시지 |
|-----|--------|
| `UNAUTHORIZED` | 로그인이 필요해요 |
| `FORBIDDEN` | 이 기능은 관리자만 쓸 수 있어요 |
| `NOT_FOUND` | 찾을 수 없어요 |
| `VALIDATION_ERROR` | 입력값을 확인해 주세요 |
| `PRICEBOOK_INACTIVE` | 활성화된 단가표가 없어요 |
| `AI_SERVICE_ERROR` | AI 서비스에 문제가 생겼어요. 잠시 후 다시 시도해 주세요 |
| `ESTIMATE_LOCKED` | 발행된 견적서는 수정할 수 없어요 |
| `FILE_UPLOAD_ERROR` | 파일을 올리는 데 실패했어요 |

## 체크리스트

문구 작성 시 아래 항목을 확인하세요.

- [ ] 해요체로 썼나요?
- [ ] 능동형으로 썼나요?
- [ ] 긍정형으로 썼나요?
- [ ] 불필요한 단어가 없나요?
- [ ] 한 문장에 하나의 메시지만 담았나요?
- [ ] 소리 내어 읽었을 때 자연스러운가요?
- [ ] 다음 화면/행동을 예측할 수 있나요?
