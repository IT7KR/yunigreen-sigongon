## 1. 문서 개요

### 1.1 목적
본 문서는 시공ON 플랫폼의 각 화면별 구성요소, 입력/출력 항목, 사용자 액션을 정의합니다.

### 1.2 화면 ID 규칙

```
SCRN-[카테고리번호][순번]

카테고리:
- 0XX: 온보딩/인증
- 1XX: 프로젝트 관리
- 2XX: AI 진단
- 3XX: 일용직 관리
- 4XX: 문서 관리
- 5XX: 슈퍼 어드민
```

### 1.3 플랫폼 구분

| 약어 | 플랫폼 | 설명 |
|------|--------|------|
| W | Web | 관리자 웹 (데스크탑) |
| M | Mobile | 모바일 앱 (Android/iOS) |
| WM | Both | 웹 + 모바일 모두 |

### 1.4 UX 고려사항 (공통)

- **타겟 사용자**: 40-50대 건설업 종사자
- **현장 환경**: 장갑 착용 가능 → 큰 터치 영역 필요 (최소 44px)
- **폰트 크기**: 본문 16px 이상, 중요 정보 18px 이상
- **색상 대비**: WCAG AA 기준 이상
- **버튼 크기**: 최소 높이 48px (모바일)
