- 플랜 변경: 현재 구독 기간 종료 후 변경 적용 (예: 3/1까지 스탠다드 → 3/2부터 프리미엄)

### 6) 노무관리 - 근로자 가입 방식 변경 (핵심 변경사항)
- **기존 방식** (1/26 회의 확정): 신고자료 업로드 시 자동 회원가입
- **변경된 방식**: 알림톡 발송 → 개인정보 동의 → 회원가입 프로세스
- 상세 흐름:
  1. 관리자가 신고자료 업로드
  2. 미등록 회원 필터링
  3. 미등록 근로자에게 알림톡 발송
  4. 근로자가 개인정보 처리방침 열람 및 동의
  5. 필수 서류 업로드 (신분증, 기초안전보건교육 이수증)
  6. ID/PW 입력하여 가입 완료

> **주의 (1/26 결정 변경)**: 킥오프 2차(1/26)에서 확정된 "관리자 등록 시 자동 계정 생성" 방식이 **개인정보 동의 기반 자발적 가입 방식**으로 변경됨. 07_MEETING_DECISIONS_SUMMARY.md의 AUTH/WORK 도메인 업데이트 필요.

### 7) 신고자료 Excel 자동 생성

- 시스템에 등록된 정보 기반 자동 입력
- **주민등록번호**: 법적 제약으로 시스템 저장 불가 → 주민번호 제외하고 다운로드

### 8) 생년월일/성별 수집 (신규)
- 근로자 가입 시 생년월일, 성별 입력 받기
- 신고자료에서 주민번호 앞 6자리 + 뒷자리 첫째(성별코드)까지 자동 입력
- 나머지 마스킹 처리 (예: 900101-1******)

### 9) 일용직 세금 계산 기준 (신규)
- 일당 15만원 이하: 비과세
- 일당 15만원 초과분: 소득세 과세
- 지방소득세 = 소득세 × 10%

### 10) 동일인 판별 기준
